# 문법

## 주석

`응아니야`뒤에 오는 문구는 전부 무시됩니다.

```
응아니야 이뒤로는 실행되지 않는 주석입니다.
새끼~야응아니야 이뒤로는 실행되지 않는 주석입니다.
```

## 숫자표기

물결표와 느낌표를 이용해서 나타냅니다. 물결표는 +1, 느낌표는 -1를 의미합니다.

```
~~ -> 2
!! -> -2
~! -> 0
!~!~~ -> 1
```

## 연산자

- +1: `~`
- -1: `!`
- 곱하기: "` `"(공백)

```
~~ -> 2
!! -> -2
~~ ~~ -> 4
~~~~ ! -> -4
```

## 변수

### 대입(거어언)

'거'와 '언' 사이의 정수번째 변수에 뒤에 오는 값을 대입합니다

```
거~언~ -> 1번째 변수에 1 지정
거~~언~ -> 2번째 변수에 1 지정
거~~~언~~ -> 3번째 변수에 2 지정
```

### 사용(ㅇ우)

'ㅇ'와 '우' 사이의 정수번째 변수를 불러옵니다.

```
ㅇ~우 -> 1번째 변수
ㅇ~~우 -> 2번째 변수
ㅇ~~~우 -> 3번째 변수
```

## 연산

변수사이에 `~`, `!`, "` `"(공백)를 넣어 각각 `+`, `-`, `*`의 용도로 사용해 변수끼리의 연산이 가능합니다.

사칙연산 순서에 상관없이 무조건 왼쪽에서부터 연산합니다.

```
거~언~
거~~언~~
ㅇ~우~ㅇ~~우 -> 3
ㅇ~우!ㅇ~~우 -> -1
ㅇ~우~ㅇ~~우 ㅇ~~우 -> 6
ㅇ~우~~ !!-> -6
```

## 콘솔

### 시발아

콘솔에서 정수를 입력받아 '시발'과 '아'사이 정수번째 변수에 대입합니다.

```
시발~아 -> 콘솔에서 입력받아 1번째 변수에 대입한다.
시발~~아 -> 콘솔에서 입력받아 2번째 변수에 대입한다.
```

### 새끼야

콘솔에 정수를 출력합니다.

```
새끼~~야 -> 콘솔에 2 출력
새끼ㅇ~우야 -> 콘솔에 1번째 변수에 저장된 정수 출력
```

### 병신

콘솔에 문자를 출력합니다. `병`과 `신`사이에 오는 정수를 ASCII 문자로 변환하여 출력합니다.

```
병~~~~~~~~~~~ ~~~~~~~~신 -> 콘솔에 X 출력

거~언~~~~~~~~~~ ~~~~~~~
거~~언~~
병ㅇ~우~ㅇ~~우신 -> 콘솔에 H 출력
```

## 지시문

### 씹덕아!

`씹덕{조건}아!{실행할 명령}`으로 작성합니다.<br>
`{조건}`이 0이라면 `{실행할 명령}`이 실행되며, 그렇지 않으면 다음줄로 넘어갑니다.

### 년

`년`뒤에 오는 정수번째 줄로 이동합니다.<br>
`년~ -> 1번째 줄로 이동`.

### 나가뒤져라

`나가뒤져라`뒤에 오는 정수를 반환하며 프로그램을 종료합니다.

## 심화 사용

해당 코드에서 입력하는 정수는 전부 변수를 대입하여 사용할 수 있습니다.

```
거~언~~~응아니야 var[1] = 3
거ㅇ~우언~~~~ ㅇ~우응아니야 var[var1] = 4 * var[1]

새끼ㅇㅇ~우우야응아니야 print(var[var1])
```
